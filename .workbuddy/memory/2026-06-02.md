## 重构图片命名规则

- 将 `worker.js` 和 `functions/upload.js` 中的图片命名从 `crypto.randomUUID()` 改为自定义 `shortId()` 函数
- `shortId()` 生成 10 位字母数字随机 ID（64^10 ≈ 1.1×10^18 组合空间）
- URL 长度从 ~40 字符缩短到 ~14 字符
- 使用 `crypto.getRandomValues()` 零 CPU 开销，避免 CF Workers 免费计划配额问题
- 已通过讨论排除了内容哈希方案（大图会吃 CF Workers CPU 配额）
- 已 commit: `212dd8c`
